**FastAPI**

**O que é?**
FastAPI é um framework web moderno, rápido e de alta performance para a construção de APIs usando Python 3.6+ baseado nas dicas de tipo padrão do Python (*type hints*). Ele é amplamente reconhecido por ser fácil de aprender, rápido de codar e por gerar documentação automática.

**Como funciona?**
O framework trabalha combinando duas ferramentas: o **Starlette** (para as partes web e roteamento) e o **Pydantic** (para a parte de dados). Quando você define os tipos de dados esperados nas rotas da API, o FastAPI automaticamente valida a entrada, serializa a saída e gera uma documentação interativa (Swagger UI e ReDoc) em tempo real, sem necessidade de código extra.
